qq {GWAF}R Documentation

function to make Qantile-Qantile (QQ) plot for p-values

Description

qq function makes the QQ plot of p-values against a uniform (0,1) distribution. The genomic control parameter for one degree freedom chi-square statistics corresponing to the p-values is also plotted.

Usage

qq(pvalue, outfile)

Arguments

pvalue P-values of interest.
outfile the file name (xxxx.bmp) for output QQ plot in bitmap format
